Evopy Documentation

Evopy to kompleksowy, ale minimalny system asystenta, który jest w stanie ewoluować z prostego skryptu konsolowego do pełnego środowiska z wieloma piaskownicami Docker. Asystent wspiera konwersję tekstu na kod Python i zarządzanie projektami w kontenerach Docker.

System Architecture

graph TD A[User] -->|Input| B(Evopy Core) B --> C{Processing Type} C -->|Code Generation| D[LLM Model] C -->|Code Execution| E[Sandbox] D --> F[Generated Code] F --> E E --> G[Execution Results] G --> B B --> H[Report Generator] H --> I[HTML/Markdown Reports]

Key Features

  • Text to Python Conversion: Generate Python code from natural language descriptions
  • Sandbox Execution: Run code safely in isolated Docker containers
  • Cross-Platform Support: Works on Linux, Windows, and macOS
  • Dependency Auto-Repair: Automatically fixes missing imports in code
  • Model Fallback: Gracefully handles unavailable models by falling back to alternatives
